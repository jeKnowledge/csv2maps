Welcome to the list2gmaps.js demo

This page was built using list2gmaps.js, a simple library that handles the creation of a Google Maps widget on a website. For this simple example we came up with a list of startups around the world. We plug the data (a simple array of locations) into list2gmaps.js and we get back a pretty map and a few other cool things.

For instance, you can scroll and pan around the map and the list of places below will be automatically updated with the currently visible locations. Plus, you get free filters for your locations which you can tick (or untick) below as well.

The library is very easy to use. In fact, feel free to read the documentation on GitHub.
